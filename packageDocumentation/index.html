Multiline Text Field

A multiline text field built in SwiftUI designed to support all availible functionality of the native single line textfield.

IRL

Standalone implementation

Implemented in a production app

Get Started 

import SwiftUI
import MultilineTextField

struct ContentView: View {
    @State var text = ""
    @State var isActive = false

    var body: some View {
      MultilineTextField(
          text: $text,
          isEditing: $isActive
      )
    }
}

MultilineTextField requires a text state, and an isActive state.

The text state stores the user entered text. The isActive state stores whether the user is actively interacting with the textfield.

Install

  1. Download for 99¢ to get the download URL
  2. Open XCode. Go to File > Swift Packges > Add Package Dependency...
  3. Select your current project
  4. Input download URL.
  5. Then click "Next".
  6. Choose Branch: master
  7. Click finish

Inputs

MultilineTextField supports a variety of supplementary inputs designed for additional functionality.

placeholder: String? = nil
Add a placeholder to your textfield.

hasBorder: Bool = true
Remove the border from the textfield. Use to either create your own custom border, or remove it altogether.

textAlignment: MultilineTextField.TextAlignment = .left
Text alignment.

placeholderAlignment: Alignment = .left
Placeholder text alignment.

textHorizontalPadding: CGFloat = 10
Horizontal text padding.

textVerticalPadding: CGFloat = 15
Vertical text padding.

placeholderHorizontalPadding: CGFloat = 15
Horizontal placeholder text padding.

placeholderVerticalPadding: CGFloat = 15
Vertical placeholder text padding.

font: UIFont = Self.defaultFont
Font.

textColor: UIColor = .label
Text color.

placeholderColor: Color = .init(.placeholderText)
Placeholder text color.

backgroundColor: UIColor = .clear
Background color.

returnType: UIReturnKeyType = .default
Custom return key.

autocorrection: Autocorrection = .default
Custom autocorrection.

autocapitalization: Autocapitalization = .sentences
Custom autocapitalization.

isEditable: Bool = true
Can users edit the textfield.

isSelectable: Bool = true
Can users select the textfield.

isScrollingEnabled: Bool = true
Can users scroll within the textfield when applicable.

... and even more not mentioned here. Download to view all customizations.

To use inputs simply add them to the initialization. 

import SwiftUI
import MultilineTextField

struct ContentView: View {
    @State var text = ""
    @State var isActive = false

    var body: some View {
      MultilineTextField(
          text: $text,
          isEditing: $isActive,
          placeholder: "Enter feedback ...",
          hasBorder: false ,
          minHeight: 150 
      )
    }
}
